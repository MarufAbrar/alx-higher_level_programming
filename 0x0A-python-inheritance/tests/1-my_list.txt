The ``Print Sorted List`` module
================================

Using ``MyList``
----------------

First import ``MyList`` from the ``1-my_list`` module:

    >>> MyList = __import__("1-my_list").MyList

Now use it:

    >>> a = MyList()
    >>> a
    []

    >>> a.append(1)
    >>> a
    [1]

    >>> a.append(4)
    >>> a.append(3)
    >>> a
    [1, 4, 3]

    >>> a.print_sorted()
    [1, 3, 4]

    >>> a.append("b")
    >>> a
    [1, 4, 3, 'b']

    >>> a.print_sorted()
    Traceback (most recent call last):
        ...
    TypeError: unorderable types: str() < int()

    >>> b = MyList()
    >>> b.append(4)
    >>> b.append(-2)
    >>> b.append(0)
    >>> b.append(3)
    >>> b
    [4, -2, 0, 3]

    >>> b.print_sorted()
    [-2, 0, 3, 4]
